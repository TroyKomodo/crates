# Crates



A collection of my open source Rust crates. PRs are welcome!

| Crate | Description |
|-------|-------------|
| [tinc](./tinc) | gRPC to REST transcoding library |
| [tinc-build](./tinc-build) | Build script for tinc protobuf annotations |
| [tinc-cel](./tinc-cel) | CEL type for validating protobuf messages |
| [tinc-pb-prost](./tinc-pb-prost) | Protobuf generated schema for tinc |
| [tinc-proc-macro](./tinc-proc-macro) | Proc macro for tinc protobuf annotations |
| [openapiv3_1](./openapiv3_1) | OpenAPI 3.1.x bindings for Rust |
| [postcompile](./postcompile) | Helper crate for post-compiling Rust code |
| [embed-changelog](./embed-changelog) | Helper macro to embed changelogs in crate docs |
| [cargo-sync-readme2](./cargo-sync-readme2) | Cargo subcommand to sync README with crate documentation |
